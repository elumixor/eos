## Notes for the reviewer

```
Eos is a minimal todo app.

HOW TO USE
1. Type a task into the bar pinned to the bottom of the home screen. Press Return to add it.
2. Type "@" inside a task to attach a project, date/time, or duration — these render as chips.
3. Drag a task between sections or days to reschedule.
4. Tap a task to edit it. Swipe left to complete or delete.

TELEGRAM SYNC (optional, off by default)
Tasks can also be created by sending a message to a Telegram bot once a user links their Telegram account from inside the app. This integration is server-side and does not affect the App Store build's behaviour. The app is fully functional without it.

PERMISSIONS
None required.

CONTACT
todoro.app.atma@gmail.com
```
