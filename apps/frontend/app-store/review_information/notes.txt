Eos is a minimal todo app.

NO DEMO ACCOUNT REQUIRED
On first launch the app bootstraps an anonymous account automatically. The reviewer can immediately create, edit, complete, reschedule, and delete tasks. To exercise sign-in, tap the avatar in the top right of the home screen and choose Google or Apple.

HOW TO USE
1. Type a task into the bar pinned to the bottom of the home screen. Press Return to add it.
2. Type "@" inside a task to attach a project, date/time, or duration — these render as chips.
3. Drag a task between sections or days to reschedule.
4. Tap a task to edit it. Swipe left to complete or delete.

TELEGRAM SYNC (optional, off by default)
Tasks can also be created by sending a message to a Telegram bot once a user links their Telegram account from inside the app. This integration is server-side and does not affect the App Store build's behavior. The app is fully functional without it.

PERMISSIONS
None required.

EXPORT COMPLIANCE
ITSAppUsesNonExemptEncryption is set to false in Info.plist. The app uses only standard HTTPS and Apple's built-in encryption.
