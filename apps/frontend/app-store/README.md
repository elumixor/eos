# App Store metadata — Eos

Drafts of every field App Store Connect asks for. Edit freely; the `/appstore-prep` command will not overwrite these once they exist.

## Layout

```
app-store/
  README.md
  categories.txt
  age_rating.md
  review_notes.md
  metadata/
    en-US/
      name.txt              ≤30
      subtitle.txt          ≤30
      promotional_text.txt  ≤170  (editable without resubmission)
      description.txt       ≤4000
      keywords.txt          ≤100  (comma-separated, no spaces around commas)
      release_notes.txt     ≤4000 (per-version "What's New")
      support_url.txt
      marketing_url.txt
      privacy_url.txt
```

## Using with App Store Connect

Manual: paste each file's contents into the matching field under your app version → App Information / Version Information.

With Fastlane: this layout is the same one [`fastlane deliver`](https://docs.fastlane.tools/actions/deliver/) expects, so you can run `fastlane deliver --metadata_path apps/frontend/app-store/metadata`.

## Screenshots

Screenshots live next to this folder under `apps/frontend/screenshots/<device>/`. They are gitignored and regenerated by `/appstore-prep`. Upload via Transporter or `fastlane deliver --screenshots_path apps/frontend/screenshots`.
